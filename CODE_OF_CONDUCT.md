# Code of Conduct
Teclab Library Information System

## Our Commitment

This project promotes a professional, respectful, and collaborative environment.

All participants — students, mentors, tutors, and contributors — are expected to engage with maturity, accountability, and mutual respect.

The objective is to create a space where ideas can be discussed critically without compromising professionalism.

---

## Expected Behavior

Participants are expected to:

- Communicate respectfully and constructively
- Provide feedback focused on ideas, not individuals
- Accept technical critique as part of engineering growth
- Encourage learning and collaboration
- Maintain professional language at all times

Disagreement is acceptable. Disrespect is not.

---

## Unacceptable Behavior

The following behaviors are not tolerated:

- Personal attacks or harassment
- Discriminatory language or conduct
- Hostile or aggressive communication
- Public shaming or intimidation
- Disruptive behavior that affects collaboration

---

## Professional Communication Principles

Engineering discussions should aim to:

- clarify problems
- improve solutions
- share knowledge responsibly
- support collective progress

Technical rigor and respectful dialogue must coexist.

---

## Enforcement

Project maintainers reserve the right to moderate discussions, reject contributions, or restrict participation when behavior violates this Code of Conduct.

Actions will be proportional and focused on restoring a healthy collaborative environment.

---

## Scope

This Code of Conduct applies to:

- repository discussions
- pull requests
- issues
- project-related communications

---

## Final Note

Professionalism is defined not only by technical ability but also by how collaboration is conducted.  
Every participant contributes to the culture of the project.
